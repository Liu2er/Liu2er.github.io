其它 ——> NSString

举例：

NSString *string = NSStringFromClass([NSArray class]);

NSLog(@"NSArray 的类名是：%@",string);

输出如下：

2015-08-18 16:10:17.908 NewCommandTest[845:137450] NSArray 的类名是：NSArray

NSNumber ——> NSString

NSNumber *number = [NSNumber numberWithInt:20];

// 转换方法1
NSString *string = [number description];

// 转换方法2
NSString *string = [NSString stringWithFormat:@"%d", [number intValue]];

本例是将@20转换为@"20"。

NSData ——> NSString

const char * ——> NSString

NSString * strNSString;

const char *pConstChar;

strNSString = [[NSString alloc] initWithUTF8String:pConstChar];

或者

strNSString = [[NSString alloc] initWithUTF8String:“字符串”];

NSString ——> const char *

或者

pConstChar = [@"字符串" UTF8String];

JSON ——> NSArray 或 NSDictionary
